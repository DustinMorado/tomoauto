Script print_header

Prints MRC standard header to screen.

This program prints the information of the standard header in an MRC file. Similar to the IMOD command header, although this one puts out a littel more information pertaining to whether or not the file is IMOD or Agard style.

Dependencies: MRC_IO_lib
