Script scale_RAPTOR_model

Changes 3dmod model file produced by RAPTOR to better erase gold.

This program takes a fiducial model generated automatically by RAPTOR and scales and changes its properties to make it better for erasing gold and for computing the alignment using IMOD tiltalign

Dependencies: MRC_IO_lib

scale_RAPTOR_model (input_filename, output_filename)
Alters 3dmod model file produced by RAPTOR to better erase gold. Scales the model to the original dataset, and makes the model point size bigger to better erase gold.

Parameters:

  • input_filename RAPTOR generated fiducial model e.g image.fid.txt
  • output_filename Output fiducial model filename e.g image.fid
