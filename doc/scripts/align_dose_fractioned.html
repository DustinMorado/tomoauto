Script align_dose_fractioned

Aligns dose-fractioned images.

This is a program to align dose-fractioned images and produce a sum. It uses IMOD to align the stack, and is a fork of a script by Chen Xu. This is useful when dosefgpu_driftcorr fails to align a stack.

align_dose_fractioned (input_filename)
Aligns dose-fractioned images. Writes IMOD command files to run xfalign, xftoxg, and newstack to generate an aligned dose-fractioned sum image.

Parameters:

  • input_filename dose-fractioned stack to align and sum e.g. 'stack.mrc'
