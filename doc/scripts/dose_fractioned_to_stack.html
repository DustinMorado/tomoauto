Script dose_fractioned_to_stack

Imitates median filter along beam axis.

This program imitates the 'image thickness' feature in the slicer window of 3dmod. It takes N slices and projects them into an average and then reassembles the averages back into a volume.

Dependencies: MRC_IO_lib

median_filter (input_filename, filter_size)
Imitates median filter along beam axis. Takes N slices and projects them into an average and then reassembles the averages into a new stack.

Parameters:

  • input_filename Input image stack e.g. 'image.rec'
  • filter_size Number of slices to average together e.g. 7
