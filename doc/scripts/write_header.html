Script write_header

Modifies the header information of a MRC file.

This program modifies and rewrites the header of an MRC file. It should be used with care as it can easily disrupt how many MRC-processing commands operate. If in doubt do not use this program. It was mainly written to handle those too often occasions when another utility erases the header or extended header of a collected tilt-series.

Dependencies: MRC_IO_lib , yago

write_header (input_filename, output_filename, options_table)
Modifies the header information of a MRC file. Reads in the options and arguments and modifies the header accordingly, writing the changes to a new MRC file.

Parameters:

  • input_filename MRC file to process e.g. 'image.mrc'
  • output_filename New MRC filename e.g. 'image_fixed.mrc'
  • options_table Table object with option flags from yago
