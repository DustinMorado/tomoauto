Module tomoauto_lib

Main tomoauto module.

This program automates the alignment of raw tilt series using the programs IMOD, RAPTOR and TOMO3D.

Dependencies: COM_file_lib

Info:

  • Release: 0.2.10
  • License: GPLv3

Functions

clean_up (input_filename, options_table)
Cleans up on successful or unsuccessful run of tomoauto. This function removes all of the intermediate files from the working directory when tomoauto fails or completes.

Parameters:

  • input_filename MRC tilt series to process
  • options_table A table object with the option flags from yago
process (input_filename, fiducial_diameter, options_table)
Main tomoauto function. This function performs the scripting and automation of processing a collected tilt series, it can simply erase hot pixels, align a tilt series or complete the reconstruction.

Parameters:

  • input_filename MRC tilt series to process
  • fiducial_diameter Size of fiducial markers in nm
  • options_table A table object with the option flags from yago
