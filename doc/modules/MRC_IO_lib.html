Module MRC_IO_lib

MRC Input Output control library.

This module provides all of the input output access for the MRC file format particularly in regards to the header information.

Functions

get_header (input_filename)
Gets standard 1024 byte MRC header. This reads the first 1024 bytes of an MRC file and returns a table object with the corresponding data. The complete header information for Agard style headers can be found here: http://www.2dx.unibas.ch/documentation/ mrc-software/fei-extended-mrc-format-not-used-by-2dx

The information for the SerialEM/IMOD style MRC file can be found here: http://bio3d.colorado.edu/imod/doc/mrc_format.txt

Parameters:

  • input_filename MRC file to read

Returns:

    header: A table with header information
is_IMOD (nint, nreal)
Checks MRC header type. This function reads the nint and nreal sections in the header and checks to see if the file comes from IMOD

Parameters:

  • nint section from header
  • nreal section from header

Returns:

    true if MRC is SerialEM/IMOD style otherwise nil
get_extended_header (input_filename)
Reads MRC file extended header. A function that reads the extended header for a given MRC file and returns a table object with the corresponnding data

Parameters:

  • input_filename MRC file to be read

Returns:

    extended_header A table object with MRC file information
get_tilt_angles (input_filename, output_filename)
Gets the tilt angles from the extended header of a tilt series. This function writes the tilt angles out to stdout or a file.

Parameters:

  • input_filename MRC file to read
  • output_filename [optional] Output file
set_header (input_filename, output_filename, header, extended_header)
Write an MRC file with a given header. This function writes a MRC file using a provided table object with header information

Parameters:

  • input_filename MRC file to set header
  • output_filename Output MRC file
  • header Table object with standard header information
  • extended_header Table object with extended header information
write_header (input_filename, output_filename, options_table)
Modifies the header information of a MRC file. This function modifes and rewrites the header of an MRC file.

Parameters:

  • input_filename MRC file to process
  • output_filename New MRC filename
  • options_table Table object with option flags from yago
get_required_header (input_filename, fiducial_diameter)
Gets all header information needed for tomoauto. This function returns a table object with a mix of data from the standard and the extended MRC header.

Parameters:

  • input_filename MRC stack filename
  • fiducial_diameter Size of fiducial markers in nm

Returns:

    header A table object with the required information
